Exploration des données sur les prêts prospères¶

Aperçu de l'enquête¶

Dans cette enqute, nous voulions comprendre deux choses :

  • Les facteurs qui peuvent tre utiliss pour prdire le dfaut de crdit.
  • Quels facteurs influencent la note de prosprit.

Aperçu de l'ensemble de données¶

Cet ensemble de donnes contient 113937 prts avec 81 variables sur chaque prt, pour les besoins de cette enqute, nous avons considrs les variables suivantes: Term, LoanStatus, BorrowerRate, ProsperRating (Alpha), ListingCategory (numeric), EmploymentStatus, DelinquenciesLast7Years, StatedMonthlyIncome, TotalProsperLoans , LoanOriginalAmount, LoanOriginationDate, Recommandations, Investisseurs.

# Ajustage de la forme des donnes
condition = (df_output['LoanStatus'] == 'Completed') | (df_output['LoanStatus'] == 'Defaulted') |\
                  (df_output['LoanStatus'] == 'Chargedoff')
df_output = df_output[condition]

def default_change(row):
    if row['LoanStatus'] == 'Chargedoff':
        return 'Defaulted'
    else:
        return row['LoanStatus']
    
df_output['LoanStatus'] = df_output.apply(default_change, axis=1)

# Rduction du nombre de catgories
categories = {1: 'Debt Consolidation', 2: 'Home Improvement', 3: 'Business', 6: 'Auto', 7: 'Other'}
def reduce_categorie(row):
    loan_category = row['ListingCategory (numeric)']
    if  loan_category in categories:
        return categories[loan_category]
    else:
        return categories[7]
    
df_output['ListingCategory (numeric)'] = df_output.apply(reduce_categorie, axis=1)
